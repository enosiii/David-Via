## Image Optimization:
- Recommended size for gallery images: 1200x800px
- Compress images using tools like:
  - TinyPNG (https://tinypng.com/)
  - Squoosh (https://squoosh.app/)
- Format: WebP or JPEG
- Max file size: 200KB per image
